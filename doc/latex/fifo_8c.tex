Logique de l\textquotesingle{}algorithme d\textquotesingle{}ordonnancement First-\/\+Come, First-\/\+Served (FCFS). 

Stratégie \+: le processus qui arrive le premier est le premier servi. Non-\/préemptif. Les E/S sont parallélisées.

\begin{DoxyItemize}

\item \# {\bfseries{Arrivées}} \+: Les nouveaux processus sont placés en fin de file.
\item \# {\bfseries{E/S}} \+: Mise à jour des processus en attente ; retour en file si un burst CPU suit.
\item \# {\bfseries{Élection}} \+: Si le CPU est libre, le processus en tête de file est élu.
\item \# {\bfseries{Exécution}} \+: Décrémentation du temps CPU et mise à jour du diagramme de Gantt.
\item \# {\bfseries{Statistiques}} \+: Incrémentation du temps d\textquotesingle{}attente pour les processus en file.
