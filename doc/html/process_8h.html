◆ etat_processus_t

enum etat_processus_t

États possibles d'un processus dans le cycle de vie de l'ordonnancement.

Valeurs énumérées
ETAT_NOUVEAU 

Processus chargé, mais pas encore arrivé dans le système

ETAT_PRET 

En file d'attente, prêt à être exécuté par le CPU

ETAT_EN_EXECUTION 

En train d'utiliser le processeur (CPU Burst)

ETAT_EN_ATTENTE 

En train d'effectuer une Entrée/Sortie (I/O Burst)

ETAT_TERMINE 

Exécution complètement terminée
