◆ allocMat()

etat_processus_t** allocMat ( int  nbLig,
int  nbCol 
)

Alloue dynamiquement une matrice d'états de processus.

  • La matrice est typiquement utilisée pour le diagramme de Gantt :
  • Lignes : Processus (indexés de 0 à nbLig-1).
  • Colonnes : Temps/Ticks (indexés de 0 à nbCol-1).
    Paramètres
    [in]nbLigNombre de lignes (nombre de processus).
    [in]nbColNombre de colonnes (durée totale de la simulation).
    Renvoie
    Un pointeur vers le tableau de pointeurs, ou NULL en cas d'échec.
